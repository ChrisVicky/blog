如何建立一个“版本”， git 是一个版本控制软件。一个版本就是一个 commit, 一个 commit 中需要 track 多个 changes, 每个 changes 和一个文件绑定

  ┌───────────┐     git add   ┌────────────┐    git commit  ┌───────────┐
